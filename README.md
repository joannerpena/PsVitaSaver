# PsVita Saver

This script makes your BackUp for you. It'll help you to save your savedata and trophys better than do it manually.

**ONLY WORKS IN MAC**

Coded with ***BASH***

TESTED with a **PSVITA PCH2001 with HENKAKU 3.60** and a MacBook with OSx ElCapitan.
